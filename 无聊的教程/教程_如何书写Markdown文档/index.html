markdown的书写是较为简单的。下面给出一些范例：

标题：

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

这是一级标题

这是二级标题

这是三级标题

这是四级标题

这是五级标题
这是六级标题

删除线：

<del>delete me<del/>

delete me

加粗：

**Bold**

Bold

斜体：

*Italic*

Italic

下划线：

<u>Underline</u>

Underline

    this is a tip

!!! danger this is a tip

    this is a tip

    this is a tip

!!! hint this is a tip

    this is a tip

!!! note
    this is a tip

!!! note this is a tip

!!! tip
    this is a tip

!!! tip this is a tip

!!! warning
    this is a tip

!!! warning this is a tip
